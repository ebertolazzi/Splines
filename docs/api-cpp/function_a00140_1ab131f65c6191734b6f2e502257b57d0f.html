Convert polynomial defined using Hermite base

\[ p(x) = p_0 h_0(t) + p_1 h_1(t) + p'_0 h_2(t) + p'_1 h_3(t) \]

to standard form

\[ p(x) = A t^3 + B t^2 + C t + D \]
